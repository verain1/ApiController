What is API controller in dotnet core?
Ans: API controller is a in-built filter in dotnet core. The purpose of this filter is 
     to send the data by the BODY of Http Protocol.
     ApiController as a filter introduced in dotnet core 3.0 and older version
     we use from body. 

     ApiController has 2 major features:
     a. Forced to send the data by the BODY {29-12-2021}
     b. Support Model validation. {29-12-2021}
